Surgimento da Internet

A internet começou a tomar forma na década de 1960 com a ARPANET, uma rede de computadores do Departamento de Defesa dos EUA. No início dos anos 90, Tim Berners-Lee criou a World Wide Web (WWW), introduzindo o conceito de páginas web e links. Foi nessa época que o HTML surgiu como a linguagem padrão para criar e conectar essas páginas.

Como Funciona o HTML

HTML é uma linguagem de marcação que utiliza tags para definir diferentes partes de um documento. Essas tags indicam ao navegador como exibir o conteúdo, funcionando como a estrutura básica de uma página web.

Principais Tags do HTML

  • <html>: Define o início de um documento HTML.
  • <head>: Contém metadados sobre o documento.
  • <body>: Onde o conteúdo visível da página é colocado.
  • <h1> a <h6>: Tags de cabeçalho, com <h1> sendo o mais importante.
  • <p>: Define um parágrafo de texto.
  • <div>: Usado para agrupar elementos de forma genérica.

Trabalhando com Texto

O HTML permite a formatação básica de texto. Tags como <b> para negrito, <i> para itálico e <u> para sublinhado são comuns para estilizar o texto.

Listas em HTML

As listas são uma maneira prática de organizar conteúdo. Existem dois tipos principais:

  • Listas Ordenadas (<ol>): Cada item é numerado automaticamente.
  • Listas Não Ordenadas (<ul>): Cada item é marcado com um ponto ou outro marcador.

Imagens em HTML

Para inserir imagens, utiliza-se a tag <img>. Veja um exemplo abaixo:

<img src="caminho/da/imagem.jpg" alt="Descrição da imagem">

Links em HTML

Links permitem a navegação entre diferentes páginas. A tag <a> é usada para criar links. Veja um exemplo:

<a href="https://www.exemplo.com">Clique aqui</a>
